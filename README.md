# AI Interview Coach Web Application

AI Interview Coach is a simple front-end web application that helps users practice interview questions and receive instant feedback.

## Features
- Role based interview questions (Data Analyst, Web Developer, HR)
- Difficulty selection (Easy, Medium, Hard)
- AI-style feedback generation
- Question navigation and random question option
- Progress tracking
- Local session history using LocalStorage

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage
